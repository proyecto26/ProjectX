# ProjectX UI

This library was generated with [Nx](https://nx.dev).

## Create new UI components

- Simple Components

```sh
npx nx g @nx/react:component button --project ui --path=libs/ui/src/lib/button/button
```

## Running unit tests

Run `nx test ui` to execute the unit tests via [Jest](https://jestjs.io).

## Deploy

To publish the library you can run the CLI command: `npm publish` from the artifact located in the dist directory.
